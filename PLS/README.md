# ProtoLens Server (PLS)

## Development

### Cargo Reference

See [Cargo Reference](https://doc.rust-lang.org/cargo/reference/index.html).

### Project Setup 

#### Fetch dependencies (optional)

```sh
cargo fetch 
```

#### Build binaries

```sh
cargo build
```

#### Build and Run
**Debug**
```sh
cargo run 
```
**Release**
```sh
cargo run -r 
```

#### Format using [rustfmt](https://github.com/rust-lang/rustfmt)
```sh
cargo fmt
```

#### Lint and fix warnings
```sh
cargo fix
```
